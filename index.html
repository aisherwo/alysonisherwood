Graphic Design, Animation, Interactive Experiences, 3D Design and more.

Alyson Isherwood is a creative, passionate, and hardworking individual with a drive to create unique and fun designs and creations.

Services

Adobe Creative Suite

I have had plenty of experience using multiple programs in the Adobe Creative Suite. The applications I am strongest in include: Illustrator, InDesign, Photoshop, XD, Animate, and After Effects. These tools allow me to be a multi-faceted asset and the perfect person for a variety of different projects.

3D Design

I am skilled in creating 3D objects using basic shapes in Autodesk 3DS Max. I know how to use different lighting styles and textures to create a realistic effect. I know how to export my creations and import them into Unity as well as animate and render out an image sequence.

Web Design

I can code in HTML5, CSS and javascript, and have used a number of different javascript libraries to implement interactivity. I know my way around Bootstrap and have some knowledge of wordPress. I'm always looking to expand my knowledge of web based coding and dive deeper into different frameworks to create stunning webpages.

Game Development

Game development is a passion of mine that has recently started to develop into something deeper. I am currently learning C# with Unity and have been in the process of simple arcade games. I have also used HTML5 Canvas and createJS libraries to create an side-scroller game with the help of Adobe Animate. This is an area I hope to pursue further in the future.
